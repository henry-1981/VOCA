# BrideVOCA v0.2 Stabilization & Deploy — Spec

**Date:** 2026-03-13
**Branch:** `feat/v0-2-mvp` → `main`
**Goal:** v0.2 MVP를 안정화하고 iPad 2대에 Vercel 배포까지 완료한다.

## Product Context

2명의 아이(다온, 지온)가 각자의 iPad에서 사용하는 가족 영어 단어 학습 앱.
마법 아카데미 테마, Day 기반 학습 루프: Learn → Test → Review → Character 성장.

- 대상 디바이스: iPad 2대 (landscape, PWA standalone)
- 배포 플랫폼: Vercel
- Backend: Firebase (Auth, Firestore)

## Completion Criteria

- [ ] 전체 테스트 통과 (0 failures)
- [ ] `npm run lint`, `npm run typecheck`, `npm run build` 모두 클린
- [ ] iPad Safari에서 PWA 홈 화면 설치 후 landscape 정상 동작
- [ ] Vercel에 배포 완료
- [ ] iPad 2대에서 가족 프로비저닝 → 프로필 전환 → 학습 루프 전체 검증
- [ ] 오프라인 상태에서 기본 학습 가능
